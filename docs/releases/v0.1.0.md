# v0.1.0

Initial release of `apntalk/freeswitch-xml-projection`.

This release provides a pure, framework-agnostic PHP projection library for FreeSWITCH `mod_xml_curl` directory `sip_auth` responses.

## Highlights

- Package-owned directory projection DTOs.
- Deterministic XML rendering via `XMLWriter`.
- `a1-hash` and plaintext password credential params.
- First-class FreeSWITCH not-found XML response.
- `mod_xml_curl` request parser with real Docker FreeSWITCH `sip_auth` fixture coverage.
- Request redaction helpers for sensitive SIP auth fields.
- Sanitized Docker capture provenance documentation.
- Opt-in Docker FreeSWITCH live smoke harness.
- Opt-in Docker FreeSWITCH chaos smoke harness.
- LF line-ending enforcement via `.gitattributes`.

## Scope

v0.1.0 intentionally supports directory `sip_auth` projection only.

Reverse-auth, message-count, gateway, network-list, dialplan, configuration, directory groups, framework bridges, and HTTP-auth enforcement are deferred.

## Boundary

This package has no Laravel, Symfony, PSR-7, APNTalk-core, database, HTTP client, ESL, provisioning, or authority dependency.

## Verification

- `composer check`
