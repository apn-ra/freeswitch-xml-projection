# Roadmap

After `v0.1.x`, future slices may add:

- reverse-auth response projection
- message-count response projection
- gateway or network-list projection once proven necessary
- richer fixture provenance from real redacted captures
- optional bridge packages for Laravel or other frameworks

Not planned for this package core:

- APNTalk authority logic
- database access
- HTTP authentication middleware
- FreeSWITCH Event Socket behavior
- provisioning workflows
- dialplan rendering
- gateway config rendering
